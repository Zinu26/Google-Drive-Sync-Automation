This repository contains a PowerShell automation tool for:

•Syncing local folders and files to Google Drive
•Detecting deleted local files and archiving them in Drive
•Automatically uploading new or modified files
•Sending email notifications with logs
•Handling OAuth2, refresh tokens, and token expiration
•Detailed processing logs for each run


⚠️ Security Notice

This project is designed to work with external configuration files.
No sensitive credentials should be stored in the script.
